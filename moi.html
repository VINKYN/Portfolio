Application de recherche et suggestion de séries

Lors de ce projet de création d'un moteur de recherche et de suggestion de séries à partir de sous-titres, la première étape consistait à rendre les données utilisables. J'ai utilisé différentes méthodes de traitement des données en utilisant des bibliothèques Python spécialisées. Pour le système de recherche, nous avons effectué une recherche intensive sur les différents algorithmes disponibles et avons effectué des tests pour trouver celui le plus optimal pour nos données.

Nous avons donc stocké dans une base de données MongoDB les mots, avec le nombre de séries contenant chaque mot ainsi que la fréquence du mot dans chaque série. Cela nous a permis de mettre en place le calcul du TF-IDF. Nous avons développé une API FastAPI en Python pour le calcul du TF-IDF et la recherche de séries, en interrogeant la base de données MongoDB. De plus, nous avons créé un algorithme de recommandation de séries aussi basé sur le TF-IDF et les séries appréciées par l'utilisateur.

Pour la partie application Web, nous avons configuré l'application sur une machine virtuelle, permettant aux utilisateurs de tester facilement l'application. En parallèle, pour présenter les fonctionnalités de l'application, nous avons mis en place un site web en PHP Symfony.
